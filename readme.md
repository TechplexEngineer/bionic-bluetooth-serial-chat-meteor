bionic-serial-chat-meteor
=========================

Prerequisites
1. Meteor
2. Android SDK

Run on Device:
`meteor run android-device --mobile-server http://10.42.0.5:3000`

Run in emulator
`meteor run android`


Depends on:
- https://github.com/TechplexEngineer/Cordova-Plugin-Bionic-BluetoothSerial
